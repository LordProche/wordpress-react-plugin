Принцип работы: шорткод создает div с классом eight29-filters. В параметры этого блока заносятся настройки,
                которые были указаны в шорткоде. Взависимости от этих настроек выводятся те или иные компоненты.

Структура проекта:
    - /src/assets/css - стили
    - /src/assets/images - изображения и иконки
    - components - папка с реакт компонентами
    - context - папка с context api components
    - methods - папка с файлами, со вспомогательными функциями

Разбор файлов:
    -------
    /src/context
    -------
        SettingsContext.js - Хранит в себе объект настроек - находит фильтр на странице, извлекает из него параметры,
                             сохраняет эти параметры в объект. После чего создаёт контекст и передает в него объект
                             настроек. Это нужно для того, чтоб все дочерние компоненты могли обращаться к объекту
                             настроек, и его не нужно было прокижывать через пропсы
        useSettingsContext.js - хук, возвращает значение settingsContext
        DataContext.js - Создаёт контекст DataContext



    - /methods/core:
        scrollUp.js - функция скролла, принимает значение координаты по оси y и скроллит к этой координате,
                      если параметр не был передан, скрол к координате 0
